# Navigation

|                                         |                             |
| --------------------------------------- | --------------------------- |
| <kbd>space</kbd> / <kbd>right</kbd>     | next animation or slide     |
| <kbd>left</kbd>                         | previous animation or slide |
| <kbd>up</kbd>                           | previous slide              |
| <kbd>down</kbd>                         | next slide                  |
